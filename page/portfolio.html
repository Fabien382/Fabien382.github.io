Portfolio numérique


Mon portfolio numérique est un projet de fin du BTS, pour l'épreuve E6. Il résume les compétences que je posséde et les projets et les stages que j'ai réalisés.

Portfolio

Comment ai-je crée mon portfolio, avec quel outils

Conception du portfolio

Mon portfolio est fait principalement avec HTML, CSS et Javascript, j'utilise un framework CSS Bootstrap pour créer un portfolio responsive. Ainsi qu'une librairie d'animation de saisie en Javascript.

Hébergement

Mon portfolio est hébergé avec Github pages, il permet d'héberger gratuitement un site web. J'ai ensuite acheté un nom de domaine chez OVH , qui fait une redirection vers mon portfolio hébergé sur github.

Environnement de développement

Pour développer mon portfolio j'ai utilisé l'éditeur de code Visual Studio Code avec l'extension Beautify pour indenter le code. J'ai utilisé GitHub pour stocker mon projet, se qui m'as permis de travailler dessus n'importe où.

PurifyCSS Online

J'ai utilisé PurifyCSS Online pour supprimer tout le code CSS non utilisé

Pour le concevoir je me suis inspiré de divers Portfolio existant et de template, j'ai aussi utilisé des designs trouvés sur CodePen (comme la timeline visible sur l'accueil de mon portfolio)

Bilan de mon Portfolio Numérique

La création de mon Portfolio m'a permis d'approfondir mes connaissances en HTML, CSS et JS. De découvrir de nouveaux outils pour mieux développer.
Je continuerai a actualiser mon Portfolio avec mes nouveaux projets/expériences.
