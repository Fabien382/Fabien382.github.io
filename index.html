Compétence

Ce que je fais et les technologies que j'utilise

Front end

Le Front End c’est la partie du code qui est reçue par le client. Je rappelle que le client c’est notre navigateur Web. Il s’agit finalement des éléments du site web que l’on aperçoit à l’écran et avec lesquels on pourra interagir. Ces éléments sont composés de 3 langages: HTML, CSS et Javascript et peuvent etre aidés par des framework tels que Bootstrap.

Back end

Le Back End est la partie qu'on ne voit pas en tant que simple internaute. Plus précisement, c'est la partie du code qui est exécutée par le serveur ou la base de données. Elle représente une très grande partie d’un projet web. Les langages pour le Back End que j'utilise sont PHP, C#, Java, SQL.

Mobile Development

Le mobile development c'est la conception d'applications mobiles, pour développer des applications sous Android j'utilise Android Studio

Environnement de développement

Un environnement de développement intégré, ou IDE, est un logiciel de création d'applications, qui rassemble des outils de développement fréquemment utilisés dans une seule interface utilisateur graphique (GUI). J'utilise Visual Studio Code pour du Web (J'ai déjà utilisé PhpStrom), IntelliJ pour du Java(Ainsi que Eclipse), Visual Studio pour du C#.

Base de données

Une base de données (que nous nommerons BDD par commodité) est une collection d’informations organisées afin d’être facilement consultables, gérables et mises à jours. J'utilise MySQL et PostgreSQL comme systèmes de gestions de bases de données, SQL en langage servant à exploiter ses bases de données, j'ai déjà fait du PL/pgSQL et du PL/SQL

Outil de gestion

En outil de gestion j'utilise git (GitHub et GitLab) pour le développement collaboratif et le stockage de mes projets, j'ai déjà utilisé Jira, Trello et ClickUp pour de la gestion de projet et Jenkins comme outil d’intégration continue
